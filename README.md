The **Yuvarth Business Club Website** is built as a modern full-stack web application to:

# Installation Guide

## 1 Clone the Repository

```bash
git clone https://github.com/yourusername/yuvarth-business-club-website.git
```

```
cd yuvarth-business-club-website
```

---

# 2 Setup Backend

```
cd backend
npm install
```

Create `.env` file:

```
PORT=5000
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key
NEWS_API_KEY=your_news_api_key
```

Run backend server:

```
npm run dev
```
